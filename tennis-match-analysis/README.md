# Tennis Match Data Analysis – Old Dominion University

This project analyzes match statistics for the Old Dominion University Men's Tennis Team.

Aliyah S. Benson collects and analyzes match data to identify player performance trends and generate insights that can support match strategy and team development.

Tools Used
- Python
- Pandas
- Data Visualization
- Statistical Analysis

Objectives
- Analyze match performance data
- Identify trends in player performance
- Create visualizations for coaching insights
