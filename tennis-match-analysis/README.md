# ODU Men's Tennis Match Data Analysis

This project analyzes match statistics from the Old Dominion University Men's Tennis Team. 
Player names have been anonymized to protect athlete privacy.

Aliyah S. Benson collects and analyzes match data to identify player performance trends and generate insights that support match strategy and team development.

Tools Used
- Python
- Pandas
- Data Visualization
- Statistical Analysis

Objectives
- Analyze match performance data
- Identify trends in player performance
- Create visualizations for coaching insights
